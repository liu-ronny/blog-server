## Overview

This is a personal blog application that includes a content management dashboard. The frontend was built using Next.js, and the backend was built using Node.js and MongoDB.

## Contents

This repository contains the backend code for the application. The frontend code is hosted [here](https://github.com/liu-ronny/blog-client). While the frontend is currently a work-in-progress, the backend is fully functional. The code in this repository has been documented and tested.
